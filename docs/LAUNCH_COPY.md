## Short launch post

```text
TradeNodeX AI Automated Trading is an open-source crypto bot control plane for developers, quant traders, and operators.

It includes WebSocket/HTTP signal ingestion, single-account fast-copy architecture, account-level risk budgets, rate limits, failure circuit breakers, Binance Futures Testnet validation, and controlled opt-in exchange adapter routes.

Built with FastAPI, Docker, SQLite WAL, and a terminal-style frontend.
```
