# Code of Conduct

TradeNodeX AI Automated Trading is an open-source infrastructure project. We want the community to be professional, precise, and respectful.

## Expected behavior

- Be respectful and constructive.
- Keep technical discussions evidence-based.
- Disclose uncertainty when reporting exchange behavior.
- Avoid sensational claims about trading performance.
- Protect your own and others' secrets.
- Use issue templates and provide reproducible details.

## Unacceptable behavior

- Harassment, abuse, or discriminatory language.
- Posting API keys, secrets, private credentials, or personal data.
- Promoting guaranteed returns or misleading financial claims.
- Impersonating exchanges, maintainers, or official partners.
- Encouraging unsafe deployment practices.

## Reporting

If you see unacceptable behavior or a security-sensitive disclosure, report it to the maintainers. Security-sensitive reports should follow `SECURITY.md`.

## Scope

This Code of Conduct applies to repository issues, pull requests, discussions, documentation, and community spaces related to the project.
